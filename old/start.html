Getting Started

First thing to decide, burn the iso to a dvd or use rufus to make a bootable flash drive. I highly recommend a 4gb flash drive formated to fat32 and using rufus as it provides great performance compared to the slower dvd performance.

Linux can use the DD command: dd bs=4M if=/path/to/iso of=/dev/sdx && sync Use the command blkid to find your sdx to use. You will more than likely need to be root or sudo for both commands.

MD5 Sum One should always check the md5sum of the ISO file to verify that the file you downloaded is a legit working copy. Download the iso file and the md5 file together in the same directory/folder. For linux, run the command in the directory you downloaded the files to:
md5sum -c Sabayon_Linux_Gforensic_DAILY_x86.iso.md5 (substitute name of course)

For Windows you can download a free md5sum checker like WinMD5Free and use that. Some software burning programs have a built in md5sum checker.

Booting

With most newer computers, it's simple to select a boot device. Generally you can insert the dvd or the flash drive, as the computer is rebooting or starting up, hit the F12 key to enter boot menu. This is generally done right away when the computer is first rebooting or starting up. You should even see a message that even tells you what key to hit. It might not be F12, but watch the screen and it should say something like, for boot menu push F12(may vary) or to enter setup push F2 or sometimes del key.

On older computers, you may have to push F2 (or different key) to enter setup and enable boot or change your boot order. Check your manual to see how to do that, it's generally pretty simple. Just becareful when making bios changes. You can see in this video at the 13:00 mark of them changing the bios settings. Youtube has good videos on burning iso images, booting a flash drive, so don't be afraid to search youtube for a video.

For the sake of simplicity on my newer computer. I plug in my flash drive, restart, as soon as I see the boot splash, I hit F12 and a box comes up to let me select my USB flash drive. It's just that simple, nothing to change in bios or anything.
